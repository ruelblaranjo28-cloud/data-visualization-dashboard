# 🚗 Automobile Data Visualization Project  
### Exploratory Data Analysis and Visualization  
By Ruel Laranjo



## 📌 Introduction
This project analyzes automobile sales data to understand trends over time, especially during recession periods.

The goal is to:
- Clean and prepare the dataset
- Perform exploratory data analysis (EDA)
- Create meaningful visualizations
- Identify patterns in automobile sales

## 📦 Import Libraries
We import the required libraries for data manipulation and visualization.

## 📂 Load Dataset
The dataset is loaded from an online source.

## 🧹 Data Cleaning
Column names are cleaned for consistency and ease of use.

## 🔄 Data Preparation
We group the data to calculate average automobile sales per year.

## 📈 Average Automobile Sales by Year
This line chart shows how automobile sales change over time.

## 📊 Sales by Vehicle Type
This chart compares sales across different vehicle categories.

## 🗺️ Geographic Visualization
A choropleth map is used to show automobile sales distribution by location.


## ✅ Conclusion

From the analysis:
- Automobile sales fluctuate over time
- Recession periods impact sales significantly
- Certain vehicle types perform better than others
- Geographic distribution shows variation in sales

This project demonstrates how data visualization helps in understanding business trends.
